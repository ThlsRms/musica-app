# musica-app

### Formato del fichero JSON:
````Json
[
    {
         nombre: 'Canción', 
         artista: 'Artista', 
         año: 0000
    }
]
````

## Comandos Yargs:
    * **add** **-t**="Canción" **-a**="Artista" **-y**="Año"
    * **read** **-t**="Canción"
    * **editartist** **-t**="Canción" **-a**="Artista"
    * **del** **-t**="Canción"
    * **sort** **-c**="Nombre | Año"
